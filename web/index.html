Welcome to the Chat!

To join the chat, follow these steps:

  1. Open a WebSocket connection to socket.3ck0.com.
  2. Specify the chat name as a query parameter, e.g., ?room=general.
  3. Send and receive messages using the WebSocket connection.

Setting Up Your Own Server

If you want to set up your own chat server using this software, follow these steps:

  1. Clone the repository and navigate to the project directory.
  2. Modify the config.json file to configure the server settings.
  3. Run the server using the command go run main.go.
  4. Open a WebSocket connection to your server's URL.
  5. Specify the chat name as a query parameter, e.g., ?room=general.
  6. Send and receive messages using the WebSocket connection.
